Ejercicio 5 combinar clases

Ejemplos realizar un selector de clase llamado rojo y otro fondo con color de fondo en negro

Ejemplo: Definir dos clases .aviso .especial {...} El anterior selector solamente selecciona aquellos elementos con un class="especial" que se encuentren dentro de
cualquier elemento con un class="aviso". Este texto tendra las caracteristicas de letra roja con fondo negro combinando las clases.
